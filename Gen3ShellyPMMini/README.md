# Gen3ShellyPMMini
   Dieses Modul ermöglicht es, die Shelly PM Mini Gen3 in IP-Symcon zu integrieren.\
   Es können die Kanäle geschaltet werden und die Messwerte werden in IP-Symcon dargestellt.
