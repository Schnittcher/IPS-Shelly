# ShellyPlusPMMini
   Dieses Modul ermöglicht es, ein Shelly Plus PM Mini in IP-Symcon zu integrieren.\
   Es können die Kanäle geschaltet werden und die Messwerte werden in IP-Symcon dargestellt.
