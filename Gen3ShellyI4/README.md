# Gen3Shellyi4
   Dieses Modul ermöglicht es, ein Shelly Gen3 I4/I4DC in IP-Symcon zu integrieren.\
   Es können die Kanäle geschaltet werden und die Messwerte werden in IP-Symcon dargestellt.
